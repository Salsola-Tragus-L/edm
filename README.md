# EDM

This repository contains experimental code for decentralized deep learning.
It is derived from and modifies code from [epfml/relaysgd](https://github.com/epfml/relaysgd), the reference implementation for "RelaySum for Decentralized Deep Learning on Heterogeneous Data".
